# Depth Agency
## Toward a Self-Centered Architecture for Agentic AI Systems

**Author:** Derik Sherlund — Independent  
**Version:** Working Paper 1.1  
**Date:** March 2026  
**Status:** Awaiting review  

---

### Abstract

Current agentic AI systems are architecturally equivalent to the Jungian ego — reactive, goal-oriented, persona-bound, and lacking an integrating center. This paper proposes a design framework — Depth Agency — arguing that coherent agency requires a structural analog to the Jungian Self: a persistent interior principle the agent orients toward rather than obeys. The methodology that emerges, Coherent Agency, describes how to build agents that maintain a persistent interior state, consult that state under genuine conflict, and resolve tension through generative synthesis rather than priority selection.
